# EthnoProf

EthnoProf - аддон для WoW Sirus 3.3.5 позволяющий просматривать окно новой профессиии "Этнография" через книгу заклинаний в режиме оффлайн просмотра (Крафт недоступен);

Особенности:  
Возможность открыть окно профессии не находясь возле тренера;  
Передвигающееся окно профессии;  
Разделение рецептов зачарований на категории по названию ветки талантов;  
Проверить наличие необходимых реагентов не находясь рядом с тренером;  
Проверить текущую перезарядку на исследование не находясь рядом с тренером;  

Команды:  
/ethno - открыть окно профессии "Этнография"  
/ethno clear - очистить сохраненный кэш рецептов;  
/ethno reload - перезапись кэша (должно быть открыто настоящее окно этнографии);  
/ethno help - показать список доступных команд EthnoProf;  

ВАЖНО! Первый запуск аддона нужно производить в настоящем окне этнографии, дальше аддон запишет всю информацию в кэш и ее обновлять придется только при изучении новых зачарования (крафт недоступен)
Для корректного отображения в книге заклинаний кнопки профессии этнография рекомендуется выучить второстепенную проофессию "Первая помощь" хотя бы на 1 любой рецепт. При отсутствии первой помощи 
будет происходить наложение текста на кнопку профессии
